- Vision, mission, and product documentation
- Technical architecture documentation
- OpenDBC integration specification and adapter layer design
- Decoder Adapter Layer interface definition
- CAN firewall rule engine specification
- Threat detection engine specification
- Predictive maintenance engine specification
- Attack simulation engine specification
- Trust model and threat model documentation
- Security model and authentication specification
- Configuration file schemas for all subsystems
- AI-assisted development prompt library
- Sprint and decision tracking memory system
- Hardware architecture documentation for ESP32 gateway
- UI/UX guidelines and user flow documentation
- API specification for gateway ↔ app communication protocol
- Database schema for local SQLite storage
- Testing strategy and deployment documentation
- Investor and hackathon FAQ documents
- Roadmap through Q4 2027

- Defined OpenDBC boundary rules protecting upstream dependency integrity
- Encoded offline-first and privacy-by-default requirements in product constitution
